this simulation creates a Fractal which qualities can be customised by the user.
Explanation of the values:

Size: the size of the array used to store meshes

Element 0, 1, 2 etc: Here we can choose which meshes will be used in the simulation
(meshes re chosen at random)

Material: material used with the meshes, Diffuse used by default.
Depth: the starting depth of the simulation, 0 by default

Max Depth: maximum depth that can be used. 
Its not recommended to go above depth of 7 on weaker machines

Scale: the scale of children compared to the parent

Spawn chance: the chance for the child to spawn

max speed: the maximum speed with which the elements can rotate

max twist: maximum speed the elements can twist 

Spawn delay: delay in seconds between child spawns
